Logistic Regression

Classification

逻辑回归(Logistic Regression)其实是分类(Classification)问题，由于历史原因叫做”回归“。

分类问题:

简而言之，就是判断是什么的问题。

我们首先考虑二分类问题，即”是“还是”不是“。

即 $ y \in {0,;1} $ ，其中 ‘0’ 代表不是这一类，‘1’ 代表是这一类。

例如，我们判断一个肿瘤是不是恶性肿瘤，‘0’ 代表不是，‘1’ 代表是。

这时 $ h_\theta(x) $ 可以大于1 或者小于0。

逻辑回归: $ 0 \leq h_\theta(x) \leq 1 $

Logistic Regression Model

要想使

0≤hθ(x)≤10 \leq h_\theta(x) \leq 1 0≤hθ​(x)≤1

我们可以令

hθ(x)=g(θTx)h_\theta(x) = g(\theta^Tx) hθ​(x)=g(θTx)

其中，

g(z)=11+e−zg(z) = \dfrac{1}{1+e^{-z}} g(z)=1+e−z1​

$ g(z) $ 的图像是

所以，满足了 $ 0 \leq h_\theta(x) \leq 1 $ 。

$ g(z) $ called Sigmoid function or Logistic function

函数输出解释

$ h_\theta(x) $ 是输入为 $ x $ 时，$ y=1 $ 的可能性。

eg: 在患不患癌的问题上，如果 $ h_\theta(x) = 0.7 $ ，那么我们就说患者患癌的概率为0.7。

表示为 $ h_\theta(x) = P(y = 1|x;\theta) $ 。

Decision boundary（决策边界）

suppose predict $ "y = 1" $ if $ h_\theta(x) \geq 0.5 $ ,we can get $ \theta^Tx \geq 0 $ from the above image.

suppose predict $ "y = 0" $ if $ h_\theta(x) < 0.5 $ ,we can get $ \theta^Tx < 0 $ from the above image.

对于 $ h_\theta(x) = g(\theta_0 + \theta_1x_1 + \theta_2x_2) $ 我们给出任意的 $ \theta $ ，都有一条直线将平面区域分成两块，一块代表 $ h_\theta(x) \geq 0 $ ，一块代表 $ h_\theta(x) < 0 $ ，这条线我们称作决策边界(Decision Boundary)。

Non-linear decision boundary

对于 $ h_\theta(x) = g(\theta_0 + \theta_1x_1 + \theta_2x_2 + \theta_3x_1^2 + \theta_4x_2^2) $ ，我们给出任意的$ \theta $ ，会得到一个封闭的图形，它是非线性的决策边界(Non-linear decision boundary)，对于更高维$ h_\theta(x) $ 也是如此。

如何选择 $\theta $ ?

我们知道， 在Linear regression 中 $ J(\theta) = \dfrac{1}{2m}\sum\limits_{i=1}{m}(h_\theta(x{(i)}) - y{(i)})2 $ ，如果我们沿用这个表达式，会导致J(θ)J(\theta)J(θ) 是非凸的(non-convex)，因为我们的hθ(x)h_\theta(x)hθ​(x) 的表达式发生了变化。

所以，我们采用下面的cost function:

Cost(h\_\theta(x), y) = \begin{cases} -log(h\_\theta(x)) &, y = 1 \\\ \ -log(1-h\_\theta(x)) &, y = 0 \end{cases}

Multi-class classification: One-vs-all

通常情况下，我们需要做多分类，这时，我们可以借鉴二分类的思想，例如：当我们要区分第一类时，我们只需考虑哪种样本是第一类，哪种不是就可以了；区分第二类同理，以此类推，我们就可以做到多分类。

我们需要对每个类做一次逻辑回归，来得出hθ(i)(x)  (i代表第i类)h_\theta^{(i)}(x) \; (i 代表第i类)hθ(i)​(x)(i代表第i类) 用来预测样本是第iii 类的可能性。

对于一个新的输入xxx，我们需要选出 max⁡_ih_θ(i)(x)\max\_{i} h\_\theta^{(i)} (x)max_ih_θ(i)(x) ，那么xxx 就是第 iii 类。

Overfitting(过拟合)

过拟合：对于给定的训练集，函数拟合的非常好，但对于新的输入，不能给出很好的预测。


\

Addressing overfitting:

  1. 减少特征数量

    • 人工选择哪些特征留下
    • Model selection algorithm

  2. 正则化(Regularization)

    • 保留所有特征，但减少参数θj\theta_jθj​ 的值
    • 当有很多特征时，也可以很好的起作用
