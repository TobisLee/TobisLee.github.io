2018-12-26 coursera ， machine learning

基础概念

监督学习(Supervised Learning)：部分数据集已有正确答案，要找到更多的正确答案(right answer)。

  • 回归(Regression)： 预测连续值的输出，如预测房价价格与房子面积的关系

  • 分类(Classification)：预测离散值的输出，如是否患癌症(0代表否，1代表是)

非监督学习(Unsupervised Learning)：只有一个数据集，没有分出类别

  • 聚类(Clustering)：将相似的数据聚在一起当作一类，如新闻聚合

  • 非聚类(Non-clustering)：如分离声音
