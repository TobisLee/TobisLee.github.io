2018-12-26 5 min read # ML

Supervised Learning

监督学习：部分数据集已有正确答案，要找到更多的正确答案(right answer)。

Regression

回归： 预测连续值的输出，如预测房价价格与房子面积的关系

Classification

分类：预测离散值的输出，如是否患癌症(0代表否，1代表是)

Unsupervised Learning

非监督学习：只有一个数据集，没有分出类别

Clustering

聚类：将相似的数据聚在一起当作一类，如新闻聚合

Non-clustering

非聚类：如分离声音
