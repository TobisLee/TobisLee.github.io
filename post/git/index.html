Git使用中的问题及解决办法

2019-01-04 1 min read # git

强制让远程仓库与本地仓库同步

git push --set-upstream git@github.com:xxx.git master --force

让本地仓库与远程仓库关联

'origin' does not appear to be a git repository

解决办法:

git remote add origin git@github.com:xxx.git
git push -u origin master
