Leetcode 215. Kth Largest Element in an Array

2019-12-08 4 min read # array # leetcode # medium

Description

Find the kth largest element in an unsorted array. Note that it is the kth largest element in the sorted order, not the kth distinct element.

Example 1:

Input: [3,2,1,5,6,4] and k = 2
Output: 5

Example 2:

Input: [3,2,3,1,2,4,5,5,6] and k = 4
Output: 4

Note:
You may assume k is always valid, 1 ≤ k ≤ array's length.

Idea

  1. 将数组从小到大排序，然后返回第N−kN-kN−k 大的元素。

    TO: O(Nlog⁡N)O(N\log N)O(NlogN)

    SO: O(1)O(1)O(1)

  2. 将数组从大到小排序，然后返回第kkk大的元素。

    TO: O(Nlog⁡N)O(N\log N)O(NlogN)

    SO: O(1)O(1)O(1)

  3. 使用c++内置函数nth_element ，函数用法：nth_element(a.begin()+l, a.begin()+k, a.begin()+r) ，该函数会使a这个数组中[l, r)内的第k小的元素处在第k个位置上（相对位置），但是这个函数不保证其他元素有序[1]。

    TO: O(n)O(n)O(n)，nnn是区间长度。

    SO: O(1)O(1)O(1)

  4. 使用c++内置函数partial_sort，函数用法：

    partial_sort(a.begin()+l, a.begin()+k, a.begin()+r) ，该函数会使a这个数组[l, r) 区间的元素重新排序。

    TO: O(N∗log⁡M)O(N*\log M)O(N∗logM) ，NNN是区间距离，MMM是l,kl, kl,k之间的距离。

    SO: O(1)O(1)O(1)

  5. 使用priority_queue 模拟小根堆。

  6. 使用priority_queue 模拟大根堆。

  7. 用栈+排序。

Solution 1

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        int n = nums.size();
        std::sort(nums.begin(), nums.end();
        return nums[n-k];
    }
};

Solution 2

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        std::sort(nums.begin(), nums.end(), greater<int>());
        return nums[k-1];
    }
};

Solution 3

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        nth_element(nums.begin(), nums.begin()+k-1, nums.end(), greater<int>());
        return nums[k-1];
    }
};

Solution 4

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        partial_sort(nums.begin(), nums.begin() + k, nums.end(), greater<int>());
        return nums[k - 1];
    }
};

Solution 5

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        priority_queue<int, vector<int>, greater<int>> pq;
        for (auto num : nums) {
            pq.push(num);
            if (pq.size() > k) {
                pq.pop();
            }
        }
        return pq.top();
    }
};

Solution 6

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        priority_queue<int> pq(nums.begin(), nums.end());
        for (int i = 0; i < k-1; i++) {
            pq.pop();
        }
        return pq.top();
    }
};

Solution 7

class Solution {
public:
    int findKthLargest(vector<int>& nums, int k) {
        stack<int> st;
        sort(nums.begin(), nums.end(), greater<int>());
        for(int i=0; i<k; i++)
        {
            st.push(nums[i]);
        }
        return st.top();
    }
};
